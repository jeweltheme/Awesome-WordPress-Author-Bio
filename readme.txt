== Installation ==

*  From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'wp-author-bio.zip' file and upload it. Or,
*  Extract the zip file, copy and paste to the 'wp-conten/plugins/' directory of your WordPress Installation. 
*  Activate the plugin.
* Author Bio
